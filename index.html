Install

$ git clone git@github.com:bitaculous/favy.git favy
$ cd favy
$ bundle

Usage

  1. Replace favicon.png in the template folder with your favicon template image.
  2. Run rake

Options

See Rakefile or take a look at Favicon Maker README.
